Если хотя бы один элемент не является целым числом, то функция возвращает исходный кортеж.

```python
def tuple_sort(tpl):
    for elm in tpl:
        if not isinstance(elm, int):
            return tpl
    return tuple(sorted(tpl))

if __name__ == '__main__':
    print(tuple_sort((5, 5, 3, 1, 9)))
    print (tuple_sort((5, 5, 2.1, '1', 9)))
```

```python
def remove_first_occurrence(tup, value):
```

```python
from collections import Counter

def top_numbers(number_string):
    
    number_list = [int(num) for num in number_string]
    count = Counter(number_list)
    most_common = count.most_common(3)
    
    result_dict = dict(most_common)
    sorted_result = dict(sorted(result_dict.items()))
    return sorted_result
```

## Самостоятельная работа №4
### 
Ваш хороший друг владеет офисом со входом по электронным картам, ему нужно чтобы вы написали программу, которая показывала в каком порядке сотрудники входили и выходили из офиса. Определение сотрудника происходит по id. Напишите функцию, которая на вход принимает кортеж и случайный элемент (id), его можно придумать самостоятельно. Требуется вернуть новый кортеж, начинающийся с первого появления элемента в нем и заканчивающийся вторым его появлением включительно. Если элемента нет вовсе вернуть пустой кортеж. Если элемент встречается только один раз, то вернуть кортеж, который начинается с него и идет до конца исходного.

Входные данные: 
(1, 2, 3), 8) 
(1, 8, 3, 4, 8, 8, 9, 2), 8) 
(1, 2, 8, 5, 1, 2, 9), 8) 

Ожидаемый результат: 
0 
(8, 3, 4, 8)
(8, 5, 1, 2, 9)

```python
def find_entry_exit(sequence, employee_id):
  
    if employee_id not in sequence:
        return tuple()

    first_index = sequence.index(employee_id)
  
    try:
        second_index = sequence.index(employee_id, first_index + 1)
        return sequence[first_index:second_index + 1]
    except ValueError:
        return sequence[first_index:]

print(find_entry_exit((1, 2, 3), 8))
print(find_entry_exit((1, 8, 3, 4, 8, 8, 9, 2), 8))
print(find_entry_exit((1, 2, 8, 5, 1, 2, 9), 8))
```

В данном задании был изучен и реализован поиск определенного элемента в кортеже, а также его последующий вывод срезом.

# Особенности кортежей (tuple)
- Неизменяемость: Кортежи — это неизменяемые коллекции данных, которые позволяют хранить упорядоченные элементы. После создания кортежа его нельзя изменять — нельзя добавить или удалить элементы, что делает их подходящими для данных, которые должны оставаться неизменными на протяжении выполнения программы.
- Использование: Кортежи часто применяются для хранения данных, которые не предполагается изменять. Их можно использовать в качестве ключей в словарях, что невозможно для списков.
- Примеры: В задачах мы использовали кортежи для различных операций, таких как извлечение подстрок, работа с пользовательскими данными, удаление элементов, обработка информации о входах и выходах сотрудников.

Кортежи: Упорядоченные и неизменяемые. Могут хранить дублирующиеся элементы, но не позволяют их изменять. Это полезно для случаев, когда необходима защита данных от изменений. Кортежи, несмотря на свою неизменяемость, можно обрабатывать с помощью операций извлечения, объединения и создания новых кортежей на основе существующих.

# Особенности словарей (dict)
- Гибкость ключ-значение: Словари — это коллекции, которые хранят данные в виде пар ключ-значение. Ключи должны быть уникальными, а значения могут быть любыми объектами Python. Словари очень удобны, когда нужно ассоциировать одно значение с другим и легко извлекать данные по ключам.
- Использование: Словари полезны для структурирования данных, например, для подсчета частоты символов, хранения результатов операций, поиска по ключу и других задач, связанных с доступом к данным по уникальным идентификаторам.
- Примеры: В задачах использовались словари для подсчета наиболее часто встречающихся цифр, создания пар "ключ-значение" при работе с пользовательскими данными, а также для формирования структуры данных по типу входа-выхода сотрудников.

Словари: Неупорядоченные коллекции пар ключ-значение (до Python 3.7). Позволяют быстро получать данные по ключам. Очень полезны для хранения данных с уникальными ключами и быстрого поиска информации. Словари позволяют легко считать частоты символов, быстро извлекать данные по ключам, а также упорядочивать данные по ключам, если это требуется.

# Итог
Кортежи и словари — это базовые, но мощные коллекции Python, каждая из которых имеет свои особенности и области применения. Кортежи полезны, когда требуется неизменяемость данных, а словари незаменимы для хранения и быстрого поиска данных по уникальным ключам.
В данной работе мы научились использовать как кортежи, так и словари для разных целей: от работы с пользовательскими данными до создания эффективных алгоритмов для решения прикладных задач.
